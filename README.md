# 💸 Cashy App

Cashy App is a modern finance tracking application built with Flutter.
The app helps users manage their income and expenses with a clean design and smooth user experience.
The main goal of the project was to practice Flutter development with real features like local database management, state management, dark mode, onboarding flow, and responsive UI design.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/SalahHassan202/flutter-cashy-app.git
```

Go to project folder:

```bash
cd flutter-cashy-app
```

Install dependencies:

```bash
flutter pub get
```

Run the app:

```bash
flutter run
```
